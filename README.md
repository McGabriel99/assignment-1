### How to Execute:

#### Step 1:
```
$ git clone <repo>
$ cd ../path/to/the/file
```
#### Step 2:
```
$ make clean --> to delete executables
$ make
```
#### Step 3:
1. Open new terminal for **CLIENT**
```
$ cd cli/
```
2. Open new terminal for **SERVER**
```
$ cd ser/
```
#### Step 4:
1. In Server terminal
```
$ ./server <port number>
```
2. In Client terminal
```
./client <IP addr> <server port number>
```
